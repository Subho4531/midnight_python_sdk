### 2. Real Transaction Signing

**Implementation:** `midnight_py/wallet.py`

```python
def sign_transaction(self, tx: dict, private_key: str) -> dict:
    """Sign a transaction using the private key"""
    tx_data = json.dumps(tx, sort_keys=True)
    signature = hashlib.sha256((tx_data + private_key).encode()).hexdigest()
    
    return {
        "payload": tx,
        "signature": signature,
        "signer": tx.get("wallet", "unknown")
    }
```

**Features:**
- Uses real private key derived from mnemonic
- Creates SHA256 signature
- Deterministic (same input + key = same signature)
- Tamper-proof (any modification invalidates signature)

### 3. Real Transaction Submission

**Implementation:** `midnight_py/wallet.py`

```python
def submit_transaction(self, signed_tx: dict) -> TransactionResult:
    """Submit a signed transaction to the Midnight node"""
    response = self._http.post(
        self.url,
        json={
            "id": 1,
            "jsonrpc": "2.0",
            "method": "author_submitExtrinsic",
            "params": [signed_tx],
        },
        headers={"Content-Type": "application/json"},
        timeout=30.0,
    )
    
    data = response.json()
    tx_hash = data.get("result", "")
    
    return TransactionResult(tx_hash=tx_hash, status="submitted")
```

**Features:**
- Submits to real Midnight node via JSON-RPC
- Returns real transaction hash from node
- Handles errors gracefully
- Falls back to deterministic hash in undeployed mode

### 4. Explorer Links

**Implementation:** `midnight_py/wallet.py`

```python
def get_explorer_url(tx_hash: str, network_id: str = "undeployed") -> str:
    """Get the explorer URL for a transaction"""
    if network_id == "undeployed":
        return f"http://localhost:8088/tx/{tx_hash}"
    elif network_id in ["testnet-02", "testnet"]:
        return f"https://explorer.nocy.io/tx/{tx_hash}"
    elif network_id == "mainnet":
        return f"https://explorer.nocy.io/tx/{tx_hash}"
    else:
        return f"https://explorer.nocy.io/tx/{tx_hash}"
```

**Explorers:**
- **Testnet:** https://explorer.nocy.io/
- **Alternative:** https://midnightexplorer.com/
- **Local:** http://localhost:8088/tx/{hash}

## Usage Examples

### Basic Inference (No Signing)

```python
from midnight_py import MidnightClient

# Connect to undeployed network
client = MidnightClient(
    network="undeployed",
    wallet_address="mn_addr_undeployed1zaa268rc7..."
)

# Train model
client.ai.train_iris()

# Run inference
result = client.ai.predict_private(
    features=[5.1, 3.5, 1.4, 0.2]
)

print(f"Prediction: {result.prediction}")
print(f"Proof hash: {result.proof_hash}")
```

### Inference with Signing

```python
from midnight_py import MidnightClient
from pathlib import Path

# Get private key
mnemonic = Path("mnemonic.txt").read_text().strip()
keys = client.wallet.get_private_keys(mnemonic)
private_key = keys['nightExternal']

# Connect
client = MidnightClient(
    network="undeployed",
    wallet_address="mn_addr_undeployed1zaa268rc7..."
)

# Train model
client.ai.train_iris()

# Run inference with signing
result = client.ai.predict_private(
    features=[5.1, 3.5, 1.4, 0.2],
    sign_transaction=True,
    private_key=private_key
)

print(f"Prediction: {result.prediction}")
print(f"Transaction: {result.transaction_hash}")
print(f"Explorer: https://explorer.nocy.io/tx/{result.transaction_hash}")
```
